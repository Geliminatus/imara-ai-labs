# Imara AI Labs

Landing site for **Imara AI Labs** — a Tanzania-built AI startup creating purpose-built, tool-using agents on Claude.

## Agents

- **Jarvis** 🛡 — security engineer & penetration tester (OWASP Top 10, RBAC, scan → report → fix → verify)
- **Johnive** 🎨 — UI design agent (Next.js · Tailwind · shadcn/ui · Aceternity UI)

## Stack

Static single-page site — Tailwind (CDN) + vanilla JS scroll animations. The hero replicates the Aceternity UI MacBook-scroll effect with no build step.

## Run locally

Open `index.html` in a browser, or serve it:

```bash
npx serve .
```
